Getting Started

Clone and install:

git clone https://github.com/Ramilbey/dragonpath.git
cd dragonpath
npm install


Run development server:

npm start


Build for production:

npm run build


Run tests:

npm test

Deployment

Connect GitHub repo to Netlify

Build command: npm run build

Publish directory: build
